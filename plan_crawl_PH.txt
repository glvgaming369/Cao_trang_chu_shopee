Viết mã javascript chạy tampermonkey ở thị trường philippines : https://shopee.ph
Giao diện Tool: 
- Sử dụng OAuth google để kết nối ggsheets tương tự như file shopee_scraper.user.js
- Có bộ lọc : price_min sold_min
- Có nút Start crawl và Stop crawl
- Có hiển thị log tiến trình cào
Logic cào như sau: 
- Khi Start :
Đếm //*[@data-sqe="item"] để tạo vòng lặp qua tất cả item
bắt đầu item 1 đến hết 
cấu trúc của item: 
getText : (//*[@data-sqe="item"])[1]//*[@aria-label="promotion price"]/parent::*/div/span[2] (lấy giá )
getText : (//*[@data-sqe="item"])[1]//*[contains(text(),"sold")] (lấy lượt bán)
logic so sánh nếu price >= price_min và sold >= sold_min thì lấy item ID thuộc tính href của : (//*[@data-sqe="item"])[1]//a[1]
https://shopee.ph/TAU-GAMMA-PHI-Triskelion-TRISKELION-1968-T-shirt-819005-for-men-casual-fashion-2025-i.1660397705.25198233336?extraParams=%7B%22display_model_id%22%3A351034162495%2C%22model_selection_logic%22%3A3%7D
=> item ID : 25198233336
=> shop ID : 1660397705
Lưu ý : Có những item không có lượt bán thì bỏ qua, có những item 2K+ sold => 2000
đến item cuối vẫn xuất hiện //*[@class="shopee-icon-button shopee-icon-button--right" and @aria-disabled="false"] thì next page và tiếp tục cào
Hết tất cả các trang chúng ta sẽ lấy itemId  gọi tới : https://affiliate.shopee.ph/offer/product_offer/itemId để lấy thông tin push lên ggsheet
Cũng có thêm bộ lọc seller_commission_min để push lên ggsheet
